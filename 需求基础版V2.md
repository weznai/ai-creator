# AI内容创作平台 - 需求基础版 V2

## 一、用户管理

### 1.1 用户信息
| 字段 | 说明 |
|------|------|
| 用户号 | 唯一标识 |
| 手机号 | 账号标识 |
| 昵称 | 显示名称 |
| 密码 | 登录凭证 |

### 1.2 功能列表
| 功能 | 说明 |
|------|------|
| 注册 | 输入手机号 → 获取验证码 → 设置昵称和密码 |
| 登录 | 方式一：手机号 + 验证码<br>方式二：手机号 + 密码 |
| 登出 | 退出当前账号 |
| 修改密码 | 输入原密码 → 设置新密码 |
| 重置密码 | 手机号验证 → 设置新密码 |
| 个人信息修改 | 修改昵称、头像等 |

---

## 二、材料库

### 2.1 功能说明
| 功能 | 说明 |
|------|------|
| 浏览材料 | 查看所有图片和视频，支持分页加载 |
| 搜索材料 | 输入关键词，智能匹配相关材料 |
| 查看详情 | 点击查看详细信息 |
| 收藏 | 点击收藏按钮，保存到"我的收藏" |
| 取消收藏 | 再次点击取消收藏 |

### 2.2 材料类型
- 图片
- 视频

### 2.3 材料展示信息
- 材料预览（图片/视频）
- 电商作品
- 场景
- 地址
- 提示词

---

## 三、图片生成

### 3.1 页面布局
```
┌────────────────────────────┐
│                            │
│       生成结果展示区        │
│     （展示生成的图片）      │
│                            │
├────────────────────────────┤
│       生成配置区            │
│  （选择语言、模型、尺寸等）  │
└────────────────────────────┘
```

### 3.2 生成配置
| 配置项 | 说明 |
|------|------|
| 语言 | 选择生成语言 |
| 比例 | 选择图片比例 |
| 数量 | 选择生成张数 |

### 3.3 模型信息
每个模型展示：
- 模型名称
- 模型能力
- 使用场景
- 模型描述

### 3.4 尺寸选择
| 配置项 | 说明 |
|------|------|
| 场景 | 选择使用场景 |
| 方式 | 选择生成方式 |
| 尺寸 | nn×nn（具体像素） |

### 3.5 生成流程
1. 选择语言
2. 选择模型
3. 选择尺寸（场景、方式、像素）
4. 选择数量
5. 输入提示词
6. 点击生成
7. 等待生成完成
8. 查看结果

### 3.6 结果操作
| 操作 | 说明 |
|------|------|
| 预览 | 点击图片放大查看 |
| 下载 | 下载图片到本地 |
| 提交材料库 | 保存到"我的作品"和"材料库" |
| 重新生成 | 使用相同配置再次生成 |

### 4.2 生成配置
| 配置项 | 说明 |

### 4.3 模型信息
每个模型展示：
- 模型名称
- 模型能力
- 使用场景

## 六、后台管理

### 6.1 模型管理
管理图片生成和视频生成使用的模型

| 操作 | 说明 |
|------|------|
| 添加模型 | 新增一个可用模型 |
| 编辑模型 | 修改模型信息 |
| 启用/禁用 | 控制模型是否可用 |
| 排序 | 调整模型显示顺序 |

**模型信息：**
- 模型名称
- 模型能力
- 使用场景
- 模型描述
- 图标

### 6.2 尺寸管理
管理图片和视频生成可选的尺寸规格

| 操作 | 说明 |
|------|------|
| 添加尺寸 | 新增一个可选尺寸 |
| 编辑尺寸 | 修改尺寸信息 |
| 启用/禁用 | 控制尺寸是否可选 |
| 排序 | 调整尺寸显示顺序 |

**尺寸信息：**
- 场景
- 方式
- 比例（1:1、4:3、16:9等）
- 像素（nn×nn）
- 时长（视频用）

### 6.3 作品管理
管理平台作品

| 操作 | 说明 |
|------|------|
| 作品列表 | 查看所有作品 |
| 编辑作品 | 修改作品信息 |
| 删除作品 | 删除作品 |

**作品信息：**
- 作品名称
- 作品类型（图片/视频）
- 场景属性
- 创建时间
