# AI内容创作平台 - 需求基础版

## 一、用户管理

### 1.1 用户信息
| 字段 | 说明 |
|------|------|
| 手机号 | 账号标识 |
| 昵称 | 显示名称 |
| 密码 | 登录凭证 |

### 1.2 功能列表
| 功能 | 说明 |
|------|------|
| 注册 | 输入手机号 → 获取验证码 → 设置昵称和密码 |
| 登录 | 方式一：手机号 + 验证码<br>方式二：手机号 + 密码 |
| 登出 | 退出当前账号 |
| 修改密码 | 输入原密码 → 设置新密码 |
| 重置密码 | 手机号验证 → 设置新密码 |
| 个人信息修改 | 修改昵称、头像等 |

---

## 二、图片库

### 2.1 功能说明
| 功能 | 说明 |
|------|------|
| 浏览图片 | 查看所有图片，支持分页加载 |
| 搜索图片 | 输入关键词，智能匹配相关图片 |
| 查看详情 | 点击图片查看：提示词、作者、收藏数 |
| 收藏 | 点击收藏按钮，保存到"我的收藏" |
| 取消收藏 | 再次点击取消收藏 |

### 2.2 图片展示信息
- 图片预览
- 生成提示词
- 作者昵称
- 收藏数量
- 是否已收藏

---

## 三、图片生成

### 3.1 页面布局
```
┌────────────────────────────┐
│                            │
│       生成结果展示区        │
│     （展示生成的图片）      │
│                            │
├────────────────────────────┤
│       生成配置区            │
│  （选择类型、模型、尺寸等）  │
└────────────────────────────┘
```

### 3.2 生成配置
| 配置项 | 说明 |
|------|------|
| 生成类型 | 文生图：输入文字描述生成图片<br>图生图：上传参考图生成新图 |
| 选择模型 | 从模型列表中选择，每个模型有说明 |
| 图片尺寸 | 选择比例和分辨率 |
| 生成张数 | 选择一次生成几张（1-4张） |

### 3.3 模型信息
每个模型展示：
- 模型名称
- 模型能力（支持哪些生成类型）
- 使用场景
- 模型描述

### 3.4 尺寸选择
按分组展示：
- 方形（1:1）
- 横版（16:9、4:3）
- 竖版（9:16、3:4）

每个尺寸显示比例和分辨率

### 3.5 生成流程
1. 选择生成类型
2. 选择模型
3. 选择尺寸
4. 选择张数
5. 输入提示词（文生图）或上传参考图（图生图）
6. 点击生成
7. 等待生成完成
8. 查看结果

### 3.6 结果操作
| 操作 | 说明 |
|------|------|
| 预览 | 点击图片放大查看 |
| 下载 | 下载图片到本地 |
| 提交图库 | 保存到"我的作品"和"图片库" |
| 重新生成 | 使用相同配置再次生成 |

---

## 四、视频生成

### 4.1 生成配置
| 配置项 | 说明 |
|------|------|
| 选择模型 | 从视频模型列表中选择 |
| 生成方式 | 4种方式，详见下表 |
| 视频时长 | 设置生成视频的秒数 |

### 4.2 生成方式
| 方式 | 说明 | 需要上传 |
|------|------|---------|
| 参考首尾帧 | 指定开始和结束画面 | 首帧图片 + 尾帧图片 |
| 参考多帧 | 提供多个关键帧 | 多张图片 |
| 参考图片 | 基于单张图片生成 | 1张图片 |
| 图片+视频混合 | 图片和视频结合生成 | 图片 + 视频 |

### 4.3 编辑区域
- 支持拖拽上传
- 支持点击选择文件
- 显示上传进度
- 预览已上传内容

### 4.4 生成流程
1. 选择模型
2. 选择生成方式
3. 上传对应素材
4. 设置视频时长
5. 点击生成
6. 等待生成完成
7. 查看结果

### 4.5 结果操作
| 操作 | 说明 |
|------|------|
| 播放 | 在线播放视频 |
| 下载 | 下载视频到本地 |
| 保存作品 | 保存到"我的作品" |

---

| 编辑分类 | 修改分类名称 |

### 5.3 我的收藏
| 功能 | 说明 |
|------|------|

| 查看详情 | 查看图片详情和提示词 |

## 六、后台管理

### 6.1 模型管理
管理图片生成和视频生成使用的模型

| 操作 | 说明 |
|------|------|
| 添加模型 | 新增一个可用模型 |
| 编辑模型 | 修改模型信息 |
| 启用/禁用 | 控制模型是否可用 |
| 排序 | 调整模型显示顺序 |

**模型信息：**
- 模型名称
- 模型能力（文生图/图生图等）
- 使用场景说明
- 模型详细描述
- 图标

### 6.2 尺寸管理
管理图片生成可选的尺寸规格

| 操作 | 说明 |
|------|------|
| 添加尺寸 | 新增一个可选尺寸 |
| 编辑尺寸 | 修改尺寸信息 |
| 启用/禁用 | 控制尺寸是否可选 |
| 排序 | 调整尺寸显示顺序 |

**尺寸信息：**
- 所属分组（方形/横版/竖版）
- 比例（如 1:1、16:9）
- 分辨率（如 1024×1024）
