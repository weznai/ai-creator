# AI创作平台 - 需求规格文档

## 1. 项目概述

### 1.1 项目背景
构建一个AI创作平台，支持图片生成、视频生成、文案创作、作品管理等功能，并支持一键发布到小红书、抖音、快手等主流平台，为用户提供便捷的AI内容创作与分发工具。

### 1.2 核心功能
- **图片生成**：文生图、图生图，支持多种AI模型
- **视频生成**：多帧参考、图片视频混合生成
- **文案创作**：多场景文案生成、智能优化
- **作品管理**：作品分类、收藏管理
- **平台发布**：小红书（试点）、抖音、快手一键发布

### 1.3 技术栈（待确认）
- 前端：React/Vue
- 后端：Node.js/Java/Python
- 数据库：MySQL/PostgreSQL
- 存储：OSS/MinIO
- AI服务：OpenAI/Google/DashScope/Replicate

## 2. 用户管理模块

### 2.1 用户实体
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | Long | Y | 用户ID |
| phone | String(11) | Y | 手机号，唯一 |
| nickname | String(50) | N | 昵称 |
| password | String(255) | N | 密码（加密存储） |
| avatar | String(255) | N | 头像URL |

| created_at | DateTime | Y | 创建时间 |
| updated_at | DateTime | Y | 更新时间 |

### 2.2 功能列表

#### 2.2.1 注册
- **流程**：手机号 → 发送验证码 → 验证 → 设置昵称和密码 → 注册成功
- **接口**：
  - `POST /api/auth/send-code` - 发送验证码
  - `POST /api/auth/register` - 注册
- **参数**：
  ```json
  {
    "phone": "13800138000",
    "code": "123456",
    "nickname": "用户昵称",
    "password": "password123"
  }
  ```
- **验证规则**：
  - 手机号格式校验
  - 验证码6位数字，5分钟有效
  - 密码8-20位，包含字母和数字
  - 昵称2-20字符

#### 2.2.2 登录
- **登录方式**：
  1. 手机号 + 验证码登录
  2. 手机号 + 密码登录
  
- **接口**：
  - `POST /api/auth/login` - 登录
  - `POST /api/auth/send-code` - 发送验证码（复用）
  
- **参数**：
  ```json
  // 密码登录
  {
    "phone": "13800138000",
    "password": "password123",
    "loginType": "password"
  }
  // 验证码登录
  {
    "phone": "13800138000",
    "code": "123456",
    "loginType": "code"
  }
  ```
- **返回**：
  ```json
  {
    "token": "jwt_token",
    "expiresIn": 7200,
    "user": {
      "id": 1,
      "phone": "138****8000",
      "nickname": "用户昵称",
      "avatar": "https://..."
    }
  }
  ```

#### 2.2.3 登出
- **接口**：`POST /api/auth/logout`
- **逻辑**：清除Token，客户端删除本地存储

#### 2.2.4 修改密码
- **前置条件**：已登录
- **接口**：`PUT /api/user/password`
- **参数**：
  ```json
  {
    "oldPassword": "password123",
    "newPassword": "newPassword456"
  }
  ```

#### 2.2.5 重置密码
- **前置条件**：未登录
- **流程**：手机号 → 验证码 → 设置新密码
- **接口**：`POST /api/auth/reset-password`
- **参数**：
  ```json
  {
    "phone": "13800138000",
    "code": "123456",
    "newPassword": "newPassword456"
  }
  ```

#### 2.2.6 用户信息修改
- **接口**：`PUT /api/user/profile`
- **参数**：
  ```json
  {
    "nickname": "新昵称",
    "avatar": "https://..."
  }
  ```

  ```
  modelId: 模型筛选
  ```
- **返回**：
  ```json
  {
    "list": [
      {
        "id": 1,
        "title": "图片标题",
        "prompt": "提示词内容",
        "imageUrl": "https://...",
        "authorId": 1,
        "authorName": "作者",

    ],

    "page": 1,

  }
  ```

- **接口**：`GET /api/images/{id}`

#### 3.3.3 收藏/取消收藏
- **接口**：
  - `POST /api/images/{id}/collect` - 收藏
  - `DELETE /api/images/{id}/collect` - 取消收藏

#### 3.3.4 RAG检索实现方案
```
方案1：向量数据库（推荐）
- 使用Milvus/Pinecone/Weaviate存储图片prompt向量
- 检索时将query转为向量，进行相似度搜索
- 返回top-k结果

方案2：Elasticsearch + 向量插件
- 使用ES的dense_vector字段类型
- 支持混合检索（关键词 + 向量）

方案3：数据库 + Embedding API
- 存储prompt的embedding向量
- 使用余弦相似度计算
```

## 12. 非功能性需求

### 12.1 性能要求
- 图片列表加载 < 1s
- 生成任务创建 < 500ms
- 发布任务创建 < 500ms
- 支持并发1000用户

### 12.2 安全要求
- 密码bcrypt加密
- JWT Token认证，2小时过期
- 敏感接口限流
- XSS/CSRF防护
- 平台Token/Cookie加密存储
- 发布接口权限校验

### 12.3 存储要求
- 图片支持WebP格式
- 图片压缩质量可配置
- 支持CDN加速
- 视频支持H.264编码

### 12.4 发布平台要求
- Token自动刷新机制
- 发布失败自动重试（最多3次）
- 定时发布精确到分钟
- 多平台发布支持事务回滚
